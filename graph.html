Graph Types — Quick Reference

DIRECTED (Digraph)
Edges have direction (u→v ≠ v→u). Examples: Web links, DAGs, dependency graphs, Twitter follows.
UNDIRECTED
Edges are bidirectional. Examples: Facebook friends, road networks, islands connectivity.
WEIGHTED
Each edge has a cost/distance. Use Dijkstra, Bellman-Ford, Prim, Kruskal.
DAG
Directed Acyclic Graph. Enables topological sort. Course schedule, build deps, task ordering.
⚡ Interview Key: Always clarify graph type before coding. Ask: directed? weighted? cycles possible?
SPARSE vs DENSE
Sparse: E ≈ O(V). Use adjacency list.
Dense: E ≈ O(V²). Matrix may be fine.
Rule: Grid graphs are always sparse.
CONNECTED COMPONENTS
Group of nodes where all are reachable from each other. Find using BFS/DFS or DSU. Count = islands problem.
DEGREE
In-degree: edges coming in.
Out-degree: edges going out.
Euler path: exactly 2 nodes odd degree.
Topological: start from in-degree 0.
BIPARTITE
Can color graph with 2 colors such that no adjacent nodes share color. Check: BFS/DFS 2-coloring. No odd cycles.
TREE AS GRAPH
Tree = undirected connected graph with V-1 edges and no cycles. Useful: treat tree problems as graph problems.
GRID GRAPH
2D matrix as graph. Each cell = node. 4-directional or 8-directional edges. BFS for shortest path in unweighted grid.

// BFS Template — Python
from collections import deque

def bfs(graph, start):
    visited = {start}
    queue = deque([start])
    dist = {start: 0}
    
    while queue:
        node = queue.popleft()
        
        for nbr in graph[node]:
            if nbr not in visited:
                visited.add(nbr)
                dist[nbr] = dist[node] + 1
                queue.append(nbr)
    
    return dist

// DFS Template — Python (iterative)
def dfs(graph, start):
    visited = set()
    stack = [start]
    
    while stack:
        node = stack.pop()
        if node in visited: continue
        visited.add(node)
        
        for nbr in graph[node]:
            if nbr not in visited:
                stack.append(nbr)
    
    return visited

# Recursive version:
def dfs_rec(node, visited, graph):
    visited.add(node)
    for nbr in graph[node]:
        if nbr not in visited:
            dfs_rec(nbr, visited, graph)

# Multi-Source BFS Template
def multi_source_bfs(grid):
    rows, cols = len(grid), len(grid[0])
    queue = deque()
    visited = set()
    
    # Seed ALL sources at distance 0
    for r in range(rows):
        for c in range(cols):
            if grid[r][c] == 'source':
                queue.append((r, c, 0))
                visited.add((r, c))
    
    # Standard BFS from multiple starts
    while queue:
        r, c, dist = queue.popleft()
        for dr, dc in [(0,1),(0,-1),(1,0),(-1,0)]:
            nr, nc = r+dr, c+dc
            if valid(nr,nc) and (nr,nc) not in visited:
                visited.add((nr,nc))
                queue.append((nr,nc,dist+1))

# Cycle Detection Directed — 3-color DFS
# color: 0=unvisited, 1=processing, 2=done
def has_cycle(graph, n):
    color = [0] * n
    
    def dfs(node):
        color[node] = 1  # Mark as processing
        for nbr in graph[node]:
            if color[nbr] == 1: return True
            if color[nbr] == 0:
                if dfs(nbr): return True
        color[node] = 2  # Mark as done
        return False
    
    return any(dfs(i) for i in range(n)
                if color[i] == 0)
Topological D · Topological Sort — BFS (Kahn's) + DFS
Intuition
Linear ordering of nodes such that for every edge u→v, u comes before v. Only valid for DAGs.
Kahn's (BFS)
Process nodes with in-degree 0 first. Decrement neighbors' in-degrees. Intuitive for cycle detection too.
DFS Post-order
Finish DFS → push to stack. Reverse stack = topo order.
Time / Space

# Kahn's Algorithm — BFS Topo Sort
from collections import deque, defaultdict

def topo_sort(n, edges):
    graph = defaultdict(list)
    indegree = [0] * n
    
    for u, v in edges:
        graph[u].append(v)
        indegree[v] += 1
    
    queue = deque(i for i in range(n)
                  if indegree[i] == 0)
    order = []
    
    while queue:
        node = queue.popleft()
        order.append(node)
        for nbr in graph[node]:
            indegree[nbr] -= 1
            if indegree[nbr] == 0:
                queue.append(nbr)
    
    # len(order) < n → cycle!
    return order if len(order) == n else []

# Dijkstra — Python (heapq)
import heapq

def dijkstra(graph, src, n):
    dist = [float('inf')] * n
    dist[src] = 0
    heap = [(0, src)]  # (distance, node)
    
    while heap:
        d, u = heapq.heappop(heap)
        
        if d > dist[u]: continue  # stale entry
        
        for v, w in graph[u]:
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
                heapq.heappush(heap, 
                    (dist[v], v))
    
    return dist

# Bellman-Ford Template
def bellman_ford(n, edges, src):
    dist = [float('inf')] * n
    dist[src] = 0
    
    # Relax V-1 times
    for _ in range(n - 1):
        for u, v, w in edges:
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
    
    # Check negative cycle (V-th relaxation)
    for u, v, w in edges:
        if dist[u] + w < dist[v]:
            return None  # negative cycle!
    
    return dist

# Floyd-Warshall Template
def floyd_warshall(n, edges):
    INF = float('inf')
    dist = [[INF]*n for _ in range(n)]
    
    for i in range(n): dist[i][i] = 0
    for u, v, w in edges: dist[u][v] = w
    
    for k in range(n):       # intermediate
        for i in range(n):     # source
            for j in range(n): # dest
                if dist[i][k] + dist[k][j] < dist[i][j]:
                    dist[i][j] = dist[i][k] + dist[k][j]
    
    return dist

# DSU with Path Compression + Union by Rank
class DSU:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n
        self.components = n
    
    def find(self, x):
        if self.parent[x] != x:
            self.parent[x] = self.find(
                self.parent[x])  # Path compress
        return self.parent[x]
    
    def union(self, x, y):
        px, py = self.find(x), self.find(y)
        if px == py: return False  # Cycle!
        if self.rank[px] < self.rank[py]:
            px, py = py, px
        self.parent[py] = px
        if self.rank[px] == self.rank[py]:
            self.rank[px] += 1
        self.components -= 1
        return True

# Kruskal's MST
def kruskal(n, edges):
    # edges: [(weight, u, v)]
    edges.sort()  # sort by weight
    dsu = DSU(n)
    mst_cost, mst_edges = 0, []
    
    for w, u, v in edges:
        if dsu.union(u, v):   # no cycle
            mst_cost += w
            mst_edges.append((u, v, w))
        if len(mst_edges) == n-1: break
    
    return mst_cost

# Prim's MST (min-heap)
def prim(graph, n):
    visited = [False] * n
    heap = [(0, 0)]  # (cost, node)
    total = 0
    while heap:
        cost, u = heapq.heappop(heap)
        if visited[u]: continue
        visited[u] = True
        total += cost
        for v, w in graph[u]:
            if not visited[v]:
                heapq.heappush(heap,(w,v))
    return total
Bridges/AP I · Bridges and Articulation Points (Tarjan's)
Intuition
Bridge: edge whose removal disconnects graph. AP: vertex whose removal disconnects. Track discovery time & low values.
Key Insight
low[v] = min(disc[v], disc of back-edge ancestors). If low[v] > disc[u] → (u,v) is a bridge.
Time / Space
O(V + E) / O(V)
1192 Critical Connections 1568 Min Days to Disconnect 
# Bridges (Critical Connections) — Tarjan
def find_bridges(n, connections):
    graph = defaultdict(list)
    for u, v in connections:
        graph[u].append(v)
        graph[v].append(u)
    
    disc = [-1] * n
    low  = [0] * n
    timer = [0]
    bridges = []
    
    def dfs(u, parent):
        disc[u] = low[u] = timer[0]
        timer[0] += 1
        for v in graph[u]:
            if disc[v] == -1:
                dfs(v, u)
                low[u] = min(low[u], low[v])
                if low[v] > disc[u]:
                    bridges.append([u,v])
            elif v != parent:
                low[u] = min(low[u], disc[v])
    
    for i in range(n):
        if disc[i] == -1: dfs(i, -1)
    return bridges

# Bipartite Check — BFS Coloring
def is_bipartite(graph):
    n = len(graph)
    color = [-1] * n
    
    for start in range(n):  # handle disconnected
        if color[start] != -1: continue
        queue = deque([start])
        color[start] = 0
        
        while queue:
            u = queue.popleft()
            for v in graph[u]:
                if color[v] == -1:
                    color[v] = 1 - color[u]
                    queue.append(v)
                elif color[v] == color[u]:
                    return False  # Odd cycle!
    
    return True

# 0-1 BFS with Deque
def zero_one_bfs(grid, src, dst):
    dist = {src: 0}
    dq = deque([(0, src)])
    
    while dq:
        d, u = dq.popleft()
        if d > dist.get(u, float('inf')):
            continue
        
        for v, w in neighbors(u, grid):
            nd = d + w
            if nd < dist.get(v, float('inf')):
                dist[v] = nd
                if w == 0:
                    dq.appendleft((nd,v))  # front
                else:
                    dq.append((nd,v))      # back
    
    return dist.get(dst, -1)

from collections import deque, defaultdict
import heapq

# ── BFS shortest path ──────────────
def bfs(graph, start, end):
    q, visited = deque([(start, 0)]), {start}
    while q:
        u, d = q.popleft()
        if u == end: return d
        for v in graph[u]:
            if v not in visited:
                visited.add(v); q.append((v, d+1))
    return -1

# ── Dijkstra ───────────────────────
def dijkstra(graph, src, n):
    dist = [float('inf')] * n; dist[src] = 0
    heap = [(0, src)]
    while heap:
        d, u = heapq.heappop(heap)
        if d > dist[u]: continue
        for v, w in graph[u]:
            if dist[u]+w < dist[v]:
                dist[v] = dist[u]+w
                heapq.heappush(heap, (dist[v], v))
    return dist

# ── DSU ────────────────────────────
class DSU:
    def __init__(self, n):
        self.p = list(range(n)); self.r = [0]*n
    def find(self, x):
        if self.p[x]!=x: self.p[x]=self.find(self.p[x])
        return self.p[x]
    def union(self, x, y):
        a,b=self.find(x),self.find(y)
        if a==b: return False
        if self.r[a]<self.r[b]: a,b=b,a
        self.p[b]=a
        if self.r[a]==self.r[b]: self.r[a]+=1
        return True

# ── Topological Sort (Kahn's) ──────
def topo(n, edges):
    g=defaultdict(list); ind=[0]*n
    for u,v in edges: g[u].append(v); ind[v]+=1
    q=deque(i for i in range(n) if ind[i]==0)
    res=[]
    while q:
        u=q.popleft(); res.append(u)
        for v in g[u]:
            ind[v]-=1
            if ind[v]==0: q.append(v)
    return res if len(res)==n else []

import java.util.*;

// BFS
public int bfs(int[][] graph, int src, int dst) {
    Queue<int[]> q = new LinkedList();
    boolean[] vis = new boolean[graph.length];
    q.offer(new int[]{src, 0});
    vis[src] = true;
    while (!q.isEmpty()) {
        int[] cur = q.poll();
        if (cur[0] == dst) return cur[1];
        for (int v : graph[cur[0]])
            if (!vis[v]) { vis[v]=true; q.offer(new int[]{v,cur[1]+1}); }
    }
    return -1;
}

// Dijkstra
public int[] dijkstra(List<int[]>[] graph, int src) {
    int n = graph.length;
    int[] dist = new int[n];
    Arrays.fill(dist, Integer.MAX_VALUE);
    dist[src] = 0;
    PriorityQueue<int[]> pq = new PriorityQueue<>(
        (a,b) -> a[0]-b[0]);
    pq.offer(new int[]{0, src});
    while (!pq.isEmpty()) {
        int[] cur = pq.poll();
        if (cur[0] > dist[cur[1]]) continue;
        for (int[] e : graph[cur[1]]) {
            int nd = cur[0] + e[1];
            if (nd < dist[e[0]]) {
                dist[e[0]] = nd;
                pq.offer(new int[]{nd, e[0]});
            }
        }
    }
    return dist;
}

    int[] p, r;
    DSU(int n) { p=new int[n]; r=new int[n];
        for (int i=0;i<n;i++) p[i]=i; }
    int find(int x) { return p[x]==x?x:(p[x]=find(p[x])); }
    boolean union(int x, int y) {
        int a=find(x),b=find(y); if(a==b) return false;
        if(r[a]<r[b]){int t=a;a=b;b=t;}
        p[b]=a; if(r[a]==r[b]) r[a]++; return true;
    }
}

#include <bits/stdc++.h>
using namespace std;

// BFS
int bfs(vector<vector<int>>& g, int src, int dst) {
    vector<bool> vis(g.size());
    queue<pair<int,int>> q;
    q.push({src,0}); vis[src]=true;
    while(!q.empty()) {
        auto [u,d] = q.front(); q.pop();
        if(u==dst) return d;
        for(int v:g[u]) if(!vis[v]){vis[v]=1;q.push({v,d+1});}
    }
    return -1;
}

// Dijkstra
vector<int> dijkstra(
    vector<vector<pair<int,int>>>& g, int src) {
    int n=g.size();
    vector<int> dist(n, INT_MAX); dist[src]=0;
    priority_queue<pair<int,int>,vector<pair<int,int>>,
        greater<>> pq;
    pq.push({0,src});
    while(!pq.empty()){
        auto [d,u]=pq.top(); pq.pop();
        if(d>dist[u]) continue;
        for(auto [v,w]:g[u])
            if(dist[u]+w<dist[v])
                {dist[v]=dist[u]+w;pq.push({dist[v],v});}
    }
    return dist;
}

// DSU with path compression + rank
struct DSU {
    vector<int> p, r;
    DSU(int n): p(n), r(n,0) {
        iota(p.begin(),p.end(),0); }
    int find(int x){ return p[x]==x?x:p[x]=find(p[x]); }
    bool unite(int x, int y){
        int a=find(x),b=find(y); if(a==b) return false;
        if(r[a]<r[b]) swap(a,b);
        p[b]=a; if(r[a]==r[b]) r[a]++; return true;
    }
};

Interview Strategy — 5-Step Framework

Step 1 · 30 seconds
Identify Graph Type
  • Directed or undirected?
  • Weighted edges?
  • Cyclic or DAG?
  • Dense or sparse?
  • Grid or explicit graph?
  • Connected or disconnected?
Step 2 · 30 seconds
Identify Pattern
  • Shortest path → BFS/Dijkstra
  • Connectivity → DFS/DSU
  • Ordering → Topo Sort
  • Components → DFS/DSU
  • Grouping → DSU
  • Critical edges → Tarjan
Step 3 · 1 minute
Select Algorithm
  • Apply recognition sheet
  • Consider constraints (V, E size)
  • Choose representation
  • Decide BFS/DFS/heap
  • Plan data structures
  • State what you'll use verbally
Step 4 · Coding
Complexity Analysis
  • State time complexity upfront
  • State space complexity
  • Justify the choice
  • V = vertices, E = edges
  • Grid: V=R*C, E=4*R*C
  • Mention bottlenecks
Step 5 · Always
Edge Cases
  • Empty graph (n=0)
  • Single node
  • Disconnected graph
  • Self-loops
  • Negative weights (Dijkstra!)
  • src == dst already
💡 Golden Rule: In your first 2 minutes, always verbalize: "This is a [directed/undirected] [weighted/unweighted] graph problem. I'll use [Algorithm] because [reason]. Time complexity is [TC], space is [SC]." This shows pattern recognition mastery.
